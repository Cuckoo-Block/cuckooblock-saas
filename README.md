## Start On Boot

This repo includes a `systemd` unit at `systemd/cuckoo-app.service` so the app can start automatically when the machine boots.

Build and verify the app once:

```bash
npm install
npm run build
npm run start:startup
```

Install and enable the service:

```bash
sudo cp /home/admin/cuckoo-app/systemd/cuckoo-app.service /etc/systemd/system/cuckoo-app.service
sudo systemctl daemon-reload
sudo systemctl enable --now cuckoo-app.service
```

Check status and logs:

```bash
sudo systemctl status cuckoo-app.service
journalctl -u cuckoo-app.service -f
```
